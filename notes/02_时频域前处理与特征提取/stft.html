一、 傅里叶变换的局限性与语音的短时平稳性假说

时频折中

标准的快速傅里叶变换（FFT）具有极高的频域分辨率，但它假设输入信号在时间轴上是无限延伸且静态不随时间变化的。然而，人类的语音是一个高度动态的非平稳信号（Non-stationary Signal），音素、声调、声带张力都在毫秒级发生剧烈改变。如果直接对整段语音做傅里叶变换，只能得到声音包含哪些频率，却彻底丢失了这些频率发生在什么时间点的信息。

短时平稳性假说（Short-Time Stationarity Hypothesis）：
虽然语音信号整体是时变的，但在极短的时间跨度内（通常为 10ms 至 30ms），声带肌肉和声道形状来不及发生剧烈位移，可以近似将这一小段音频视作一个伪平稳信号（Quasi-stationary Signal）。短时傅里叶变换（STFT）正是基于此，通过滑窗将长信号切割成无数个平稳的微波形，再分别进行傅里叶分析。

二、 分帧、帧移与加窗（Windowing）物理机制

数字信号处理

在工程实现中，将连续音频切片为平稳短帧需要精确控制三个核心参数，并引入窗函数来压制数学截断带来的物理副作用：

1. 帧长（Frame Length / n_fft）： 每一帧包含的样本点数（在 16kHz 采样率下，25ms 对应 400 个点，工程上为了适配高效的 FFT 算法，通常向上对齐到 2 的幂次方，即取 512）。

2. 帧移（Hop Length）： 相邻两帧之间的滑窗位移量。为了保证两帧交界处的信号连续性不丢失，相邻帧必须有重叠（Overlap）。常见的配置是帧移为帧长的 1/4 或 1/2（如取 160 或 256 样本点）。

3. 频谱泄露与窗函数（Spectral Leakage）： 用矩形直接生硬截断音频帧，会在边界处制造出物理上不存在的尖锐突变。引入快速傅里叶变换后，这些突变会在频域产生假的高频谐波成分，称为频谱泄露。为了解决这一问题，必须在离散序列上乘上一个平滑的汉明窗（Hamming Window）或汉宁窗（Hann Window），将边界幅值平滑淡化至 0，迫使信号周期化，从而提升频域的阻带衰减性能。

三、 STFT 数学计算流程与振幅谱转换

算法推推导

短时傅里叶变换的离散执行数学公式为：在时间轴的每个跳跃点上，将原始信号与窗函数相乘，随后进行离散傅里叶变换（DFT）。

1. 复数线性空间（Complex Spectrum）： FFT 的输出包含实部（Real）与虚部（Imaginary），每个时频交点都是一个复数（即 $a + bi$）。它同时隐含了该频点的**振幅**（能量强弱）与**相位**（时间偏移量）。在大多数 AI 语音任务（如语音识别 ASR）中，我们通常只保留振幅信息，丢弃相位信息。

2. 线性振幅谱与功率谱： 复数绝对值的模长 $\sqrt{a^2 + b^2}$ 即为振幅谱（Magnitude Spectrogram）；其平方 $a^2 + b^2$ 则称为功率谱（Power Spectrogram）。若将这些短帧的功率谱横向依时间顺序拼接起来，就形成了直观展现声音能量“时频交织”的二维图像。

四、 核心代码实现：从原始波形到时频矩阵

代码演练

以下是使用 Python 的 librosa 和 scipy 计算音频短时傅里叶变换，并手动将复数谱提炼为线性振幅谱的工业级标准代码：

import numpy as np
import librosa

def compute_stft_spectrogram(audio_path, n_fft=512, hop_length=160):
    """
    对输入音频执行离散短时傅里叶变换(STFT)，并输出其时间-频率轴的线性振幅谱
    """
    # 1. 载入标准 ASR 音频信号 (16kHz 采样)
    y, sr = librosa.load(audio_path, sr=16000)
    
    # 2. 执行短时傅里叶变换
    # window='hann' 表示在算法内部自动为每一帧加上汉宁窗以防止频谱泄露
    # stft_complex 的输出是一个复数矩阵（二维 Numpy 数组）
    stft_complex = librosa.stft(y, n_fft=n_fft, hop_length=hop_length, window='hann')
    
    # 3. 提取振幅谱：计算复数矩阵中每一个样本点的模长
    magnitude_spectrogram = np.abs(stft_complex)
    
    # 4. 矩阵维度特征推导验证
    # 频域轴长度 = n_fft // 2 + 1 (对称消除后的奈奎斯特单边谱)
    # 时域轴长度 = 总样本点数 // hop_length + 1
    freq_bins, time_frames = magnitude_spectrogram.shape
    
    print(f"STFT矩阵计算成功 | 频域分辨率(Bins): {freq_bins} | 时域总帧数(Frames): {time_frames}")
    return magnitude_spectrogram, sr

# 调用并观察时频矩阵规格
spec_matrix, sample_rate = compute_stft_spectrogram("clean_speech.wav")

一、 基于 C++ 内存指针连续滑窗移动的时域非平稳信号流式分帧与加窗算子

C++ Core Component

理论简述： 规避一维连续长波形的全局时变非平稳性缺陷。根据准平稳状态假说，语音在 20ms 周期内声道解剖结构处于相对静止。该算子利用 C++ 向量指针移动实现严格的时间轴重叠滑窗切片，并在内存流级并行点乘离散汉宁窗（Hann Window）算子，强制迫使切片交界边缘的振幅平滑衰减至零，从源头上抹杀由于生硬数学截断引发的频谱泄露（Spectral Leakage）伪影缺陷。

二、 基于 Python 欧拉旋转矩阵正交投影的一维离散傅里叶变换(DFT)与对数振幅谱重构

Python Matrix Component

理论简述： 完成一维向高维时频图像的非线性映射。接收前置分帧完成的准平稳向量，利用 NumPy 建立复数旋转因子正交网格算子，纯手写一维离散傅里叶变换（DFT）将信号映射至复数频域空间。提取复数谱的绝对二次型模长得到线性功率谱，并通过对数分贝变换 $20 \cdot \log_{10}$ 逼近人类耳朵的响度感知非线性，最终拼接输出适配神经网络直接吞吐的二维对数振幅语谱图（Spectrogram）。
