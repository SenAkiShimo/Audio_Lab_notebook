一、 什么是波束成形及其物理空间过滤初心

物理机理

通过声源定位（SSL）算法锁定发音人的空间方位角 \(\theta\) 后，多麦克风流水线的下一步核心任务是提取并增强该方向的目标人声，同时强行压制来自其他盲区角度的突发噪声与干扰。这一过程被称为波束成形（Beamforming / 空域滤波 Spatial Filtering）。

波束成形不需要依靠任何机械旋转硬件，纯粹利用数字信号处理算法，在特定的空间方向上拉起一个虚拟的、具有高增益的“听觉探照灯”（波束 Main Lobe）。探照灯对准的方向声音被放大，而盲区方向（旁瓣 Side Lobes）的噪声能量则被强制削弱，从而在源头上大幅提升输入信号的信噪比（SNR），保障后端语音识别（ASR）或大模型编码器的输入纯净度。

二、 波的相长干涉与相消干涉数学本质

声学物理

波束成形算法能够运行的底层物理基石是波的叠加原理与干涉现象（Interference）。当多路波形在空间或数字总线中相遇时，最终的合成振幅取决于它们之间的相位差：

空域双重干涉机制：
1. 相长干涉（Constructive Interference）： 如果两路信号在时间轴上处于完全对齐（同相，Phase Difference = 0）的状态，它们的波峰与波峰相撞、波谷与波谷相撞，叠加后的总振幅将实现线性的对数叠加。若 $M$ 个通道完全同相叠加，有用信号的能量将飙升 $M$ 倍。
2. 相消干涉（Destructive Interference）： 如果两路信号在时间轴上处于错位（异相，如相位差为 $\pi$）的状态，一个通道的波峰刚好撞上另一个通道的波谷，它们的物理能量会在叠加过程中发生剧烈的彼此对消。若将盲区方向的随机噪声进行多通道错位叠加，噪声能量会因不相关性而被强行平滑和压制。

三、 延迟求和（Delay-and-Sum）经典算子链路拓扑

算法核心

延迟求和（Delay-and-Sum, DS）波束成形是历史上最经典、运行最稳定的固定波束成形算法。其运算逻辑非常纯粹，直接在时域或频域执行两步闭环：

1. 精准平移延迟（Delay）： 声波斜向射入阵列时，到达各个麦克风的空间路径差导致了各通道波形在时间轴上天生是错位的（处于非同相状态）。为了让目标方向的人声触发相长干涉，算法根据前面 SSL 算好的各通道 TDOA 值（记为 $\tau_m$），在数字序列中对每一个通道执行反向的时间平移偏置，强行把所有通道的目标发音波形校正到同一条绝对时间线上。

2. 空域对齐求和（Sum）： 将所有完成时间对齐的通道数据横向执行标量相加，并除以总通道数 $M$ 进行均值归一化。其离散执行公式为：

$$ y[n] = \frac{1}{M} \sum_{m=0}^{M-1} x_m[n + \Delta_m] $$

其中 $\Delta_m$ 为折算成离散采样点数的延迟校正量。经过这步操作，目标方向对齐的人声能量完美爆发，而由于盲区噪声的 TDOA 与 $\tau_m$ 不匹配，噪声在平移后依然是错位的，在求和时触发相消干涉而被强行平滑压制。

四、 核心代码实现：全通道时域波形对齐叠加与波束合成

代码演练

以下代码基于 NumPy 矩阵底层算子，完整还原演示了延迟求和波束成形的硬件级解算全流水线：接收多通道包含空间时延差的数字信号矩阵，利用插值算子进行精度达到分数阶样本点的时域对齐移位（Delay）、多通道空域横向标量级联求和（Sum）的完整工程仿真：

import numpy as np

def run_delay_and_sum_beamforming(multichannel_signals, target_angle_deg, mic_distance=0.02, fs=16000):
    """
    输入多通道带噪离散信号矩阵 [num_mics, samples]，模拟时域延迟求和波束成形
    target_angle_deg: 目标说话人的已知空间方位角 (SSL扫出的角度)
    mic_distance: 相邻麦克风物理间距 (米)
    fs: 系统数字采样率 (Hz)
    """
    v_sound = 340.0 # 物理声速
    num_mics, num_samples = multichannel_signals.shape
    theta_rad = np.radians(target_angle_deg)
    
    # 1. 计算每个通道相对于参考原点（第 0 个麦克风）的理论绝对空间 TDOA 时延量
    # delta_t = m  d  sin(theta) / v
    tdoa_per_mic = [ (m  mic_distance  np.sin(theta_rad)) / v_sound for m in range(num_mics) ]
    
    # 初始化对齐后的多通道矩阵
    aligned_signals = np.zeros_like(multichannel_signals)
    
    print(f"[BEAMFORMING] 启动延迟对齐流水线 | 目标增强方位: {target_angle_deg}°")
    
    # 2. 执行 Delay 算子：在时域对各通道进行反向时移对齐
    for m in range(num_mics):
        # 换算成当前系统下的离散样本点移位数 (可能包含小数)
        sample_shift = tdoa_per_mic[m]  fs
        int_shift = int(np.round(sample_shift)) # 转换为最接近的整数样本点进行移位
        
        print(f" -> 通道 Mic_{m}: 理论空间差延迟 {tdoa_per_mic[m]1e6:.2f} μs | 执行时域反向校正移位: {-int_shift} samples")
        
        # 使用 np.roll 执行环形时移对齐（工程简化仿真，实际线上常采用傅里叶频域相移插值）
        aligned_signals[m, :] = np.roll(multichannel_signals[m, :], shift=-int_shift)
        
    # 3. 执行 Sum 算子：横向跨通道空域多维叠加均值归一化
    # np.mean(..., axis=0) 将 [num_mics, samples] 坍缩提炼为单声道标准流 [samples]
    output_beamformed_signal = np.mean(aligned_signals, axis=0)
    
    print(f"[BEAMFORMING SUCCESS] 多通道空域相长干涉合成完毕。")
    return output_beamformed_signal

# 载入我们在前前篇 mic-arrays.html 中仿真生成的包含 45.0度 入射时延的 4通道矩阵数据流进行闭环压制测试

        multichannel_signals=multichannel_pcm, 
        target_angle_deg=45.0,

        fs=16000
