一、 声波的物理机械本质

物理机制

声音在物理学中属于机械纵波（Mechanical Longitudinal Wave）。它依赖介质（如空气、水）的弹性形变进行能量传播。声源物体的振动引发空气分子的疏密交替变化，形成压力扰动的传播。

1. 声速（v）： 在标准大气压、15摄氏度的空气中，声速约为 340 m/s。声速随温度和介质密度的改变而改变。

2. 波长（λ）与频率（f）： 它们满足线性几何对应关系：λ = v / f。高频声音波长短，方向性强，易被障碍物反射；低频声音波长长，衍射（绕射）能力强，在空气中衰减速度慢。

二、 复合波、基频（F0）与谐波（Harmonics）

声学特征

除了音叉产生的“纯音”（正弦波）外，人类语言中的所有声音都是复合波（Complex Wave），由多个不同频率、幅度和相位的正弦信号叠加而成。

1. 基频（Fundamental Frequency，通常缩写为 F0）： 复合波中频率最低的那种简谐成分，它直接决定了人类听觉感知到的音高（Pitch）。声带每秒钟张合的次数即为基频。

2. 谐波 / 泛音（Harmonics / Overtones）： 频率为基频整数倍（2F0, 3F0...）的成分。谐波的幅值包络分布直接构成了音色（Timbre）。AI 语音克隆技术的核心就是精确建模这些高频谐波特征。

人声频率分布标准（AI 语音建模核心区间）：
1. 男声基频范围： 85Hz 至 180Hz 左右。
2. 女声基频范围： 165Hz 至 255Hz 左右。
3. 整体频率带宽： 虽然人类声带和共鸣腔发出的高频谐波与齿音可以延伸至 10kHz 以上，但人类言语中最核心、能量最集中的语义频段集中在 300Hz 至 3400Hz（即传统电话线通过的频段）。

三、 声压（Sound Pressure）与分贝（dB）的对数机制

能量量化

声音的强弱在物理学上用声压表示，单位为帕斯卡（Pa）。人类耳朵对声音强度的感知表现为非线性对数关系，为了匹配听觉特征，工程上引入了分贝（dB）作为无量纲的相对单位。

1. 声压级（SPL，Sound Pressure Level）： 定义为相对人类听觉下限（20微帕斯卡，即 20 μPa）的对数比例：dB SPL = 20 * log10(P / P0)。其中 P0 = 2e-5 Pa。

2. 全分贝刻度（dBFS，Decibels Relative to Full Scale）： 在数字音频（AI 语音代码）中，音频振幅的最大极限值被定义为 0 dBFS。所有数字化音频的有效能量分贝值都为**负数**（如 -12 dBFS）。分贝数越接近 0，声音音量越大；越负则声音越小。

四、 核心代码实现：使用 Python 提取 F0（基频/音高）曲线

代码演练

在语音合成（TTS）和变声（VC）算法中，需要显式提取发音人的 F0 曲线，用于建模或改变声调（如男声变女声需要拉高 F0）。以下是基于标准 YIN 算法的自相关基频提取工业级实现：

import numpy as np
import librosa

def extract_voice_pitch(audio_path, target_sr=16000):
    """
    提取语音信号中的基频（F0 / Pitch 跟踪），常用于 TTS 的声调控制和变声
    """
    # 1. 载入音频
    y, sr = librosa.load(audio_path, sr=target_sr)
    
    # 2. 设置人声基频的合理物理上下限，防止杂音干扰
    # 一般人类发声基频绝不可能低于 50Hz，也极少高于 500Hz
    f0_min = 50
    f0_max = 500
    
    # 3. 使用 YIN 算法计算每一帧的基频
    # f0 返回一维数组（表示每一帧对应的基频Hz数），voiced_flag 表示该帧是清音还是浊音
    f0, voiced_flag, voiced_probs = librosa.pyin(
        y, 
        fmin=f0_min, 
        fmax=f0_max, 
        sr=sr,
        frame_length=2048,
        hop_length=512
    )
    
    # 4. 滤除未发音帧（NaN值变为0），方便神经网络矩阵处理
    f0_cleaned = np.nan_to_num(f0)
    
    # 5. 统计该音频片段的有效平均音高（排除无声期和清音期）
    valid_f0 = f0_cleaned[f0_cleaned > 0]
    mean_pitch = np.mean(valid_f0) if len(valid_f0) > 0 else 0
    
    print(f"基频追踪完成 | 语音帧数: {len(f0_cleaned)} | 有效平均音高: {mean_pitch:.2f} Hz")
    return f0_cleaned, mean_pitch

# 执行基频提取
pitch_curve, avg_f0 = extract_voice_pitch("input_voice.wav", target_sr=16000)
