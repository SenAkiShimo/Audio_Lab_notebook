Python 提取音频特征（梅尔频谱）

Python

使用 librosa 库可以极快地加载音频并提取声学特征：

import librosa
import librosa.display

# 1. 加载音频（强制指定 16kHz 采样率）
y, sr = librosa.load('test.wav', sr=16000)

# 2. 计算梅尔频谱
mel_spec = librosa.feature.melspectrogram(y=y, sr=sr, n_mels=80)
print("梅尔频谱矩阵形状:", mel_spec.shape)
📟 Terminal Output: 
>>> 梅尔频谱矩阵形状: (80, 431)
>>> 特征提取成功，耗时 0.12 秒。

⚙️ C++ 语音信号分帧算法

C++

在底层算法部署时，通常用 C++ 实现高效的指针移动分帧：

#include <iostream>
#include <vector>

void processFrame(const std::vector<float>& signal, int frameLen, int hopLen) {
    int totalSamples = signal.size();
    for (int i = 0; i + frameLen <= totalSamples; i += hopLen) {
        // 指针滑窗处理当前音频帧
        float energy = 0.0;
        for (int j = 0; j < frameLen; ++j) {
            energy += signal[i + j] * signal[i + j];
        }
    }
    std::cout << "C++ Audio Framing Done!" << std::endl;
}

Matlab 快速傅里叶变换 (FFT)

Matlab

使用 Matlab 分析声音信号的频谱分布：

[y, Fs] = audioread('speech.wav'); % 读取音频
N = length(y);
Y = fft(y); % 快速傅里叶变换
P2 = abs(Y/N);
P1 = P2(1:N/2+1);
f = Fs*(0:(N/2))/N;
plot(f, P1); % 绘制频谱图
